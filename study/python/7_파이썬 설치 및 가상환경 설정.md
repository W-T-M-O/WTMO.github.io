## **virtual env**

### **With. Window**
```
python -m venv <env_name> → create venv

<env_name>\Scripts\activate → activate venv

pip list → pkg list

pip install <pkg_name>

deactivate
```
### **With. Mac**
```
python3 -m venv <env_name> → create venv

source ./<env_name>/Scripts/activate → activate venv

deactivate
```
